

# הוספת מוצרים ברצף (Quick Add)

## הרעיון
הוספת מצב "הוספה מהירה" בדשבורד - שדה טקסט פשוט שמאפשר להקליד שם מוצר, ללחוץ Enter, והמוצר נשמר מיד. אפשר להמשיך להקליד את השם הבא בלי לעבור דף. אחר כך אפשר ללחוץ על כל מוצר ולערוך את הפרטים הנוספים.

## איך זה יעבוד
1. בדשבורד, לחיצה על כפתור ה-+ תפתח פאנל הוספה מהירה בתחתית המסך (במקום לעבור לדף טופס)
2. שדה טקסט אחד עם כפתור "הוסף" - מקלידים שם, לוחצים Enter או על הכפתור, הפריט נשמר מיד
3. השדה מתרוקן ואפשר להקליד את הפריט הבא
4. הודעת toast קצרה מאשרת כל הוספה
5. כפתור X לסגור את הפאנל כשסיימנו
6. הכפתור + הקיים (FAB) עדיין קיים - לחיצה ארוכה/רגילה תפתח את המצב המהיר, ואפשרות נוספת תוביל לטופס המלא

## מה ישתנה

### קובץ: src/pages/Dashboard.tsx
- הוספת state לפאנל הוספה מהירה (פתוח/סגור)
- הוספת קומפוננטה של שדה טקסט בתחתית המסך עם אנימציה
- שימוש ב-addItem מ-useItems לשמירה מהירה (רק עם שם)
- כפתור FAB ישתנה: לחיצה תפתח/תסגור את פאנל ההוספה המהירה, ולחיצה ארוכה תעביר לטופס המלא

---

## פרטים טכניים

### Dashboard.tsx - שינויים
- ייבוא useItems לגישה ל-addItem
- state חדש: `quickAddOpen` (boolean), `quickName` (string)
- פאנל fixed בתחתית עם Input + Button
- בלחיצת Enter/כפתור: קריאה ל-addItem.mutateAsync({ name }) ואיפוס השדה
- ה-FAB ישתנה לכפתור toggle שפותח/סוגר את הפאנל
- הוספת לינק "טופס מלא" בתוך הפאנל למעבר ל-/item/new
